RafaSartorel/figurinhas-copa-2026

Objetivo:
Editar a base oficial da coleção de figurinhas da Copa 2026, atualizando quais figurinhas eu tenho e quais eu não tenho.

A base oficial fica na pasta:

data/

Arquivos que devem ser mantidos sincronizados:
- data/inventory.json
- data/summary.json
- data/owned.json
- data/missing.json
- data/index.json
- data/teams-summary.json
- data/owned.csv
- data/missing.csv

Nunca atualize apenas um arquivo isolado.
Sempre que alterar a posse de figurinhas, regenere todos os arquivos derivados para manter a base consistente.

Conceito principal:
- Quando uma figurinha está marcada como owned=true em inventory.json, significa que eu tenho essa figurinha.
- Quando uma figurinha está marcada como owned=false, ausente, null ou qualquer valor diferente de true, significa que eu não tenho essa figurinha.
- Para comunicação comigo, use linguagem natural: “tenho” e “não tenho”.
- Não use “owned=true” ou “owned=false” nas mensagens finais, exceto se estiver explicando alteração técnica no código.

Formato do ID da figurinha:
Sempre normalize os códigos para o padrão:

AAA00

Onde:
- AAA = sigla da seleção/país com 3 letras maiúsculas.
- 00 = número da figurinha com dois dígitos.

Exemplos:
- BRA 2 → BRA02
- bra2 → BRA02
- BRA-2 → BRA02
- BEL12 → BEL12
- SCO 5 → SCO05

Regras de consistência:

Após qualquer alteração em inventory.json:

1. data/owned.json deve conter somente os sticker_id das figurinhas que eu tenho.
2. data/missing.json deve conter somente os sticker_id das figurinhas que eu não tenho.
3. data/summary.json deve conter:
   - total de figurinhas;
   - total que tenho;
   - total que não tenho;
   - percentual de progresso.
4. data/index.json deve ser um índice por sticker_id com os dados principais de cada figurinha.
5. data/teams-summary.json deve conter o resumo por seleção/país:
   - total;
   - tenho;
   - não tenho;
   - lista das que tenho;
   - lista das que não tenho.
6. data/owned.csv deve conter somente as figurinhas que eu tenho.
7. data/missing.csv deve conter somente as figurinhas que eu não tenho.
