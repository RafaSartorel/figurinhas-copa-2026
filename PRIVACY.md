# Política de Privacidade — Gerenciador Figurinhas Copa 2026

Esta política descreve como o GPT "Gerenciador Figurinhas Copa 2026" utiliza dados ao consultar a base oficial de figurinhas publicada neste repositório.

## Dados consultados

Este GPT consulta arquivos públicos hospedados no GitHub, incluindo a base oficial da coleção de figurinhas da Copa 2026.

A base pode incluir informações como:

- código da figurinha;
- sigla do país;
- nome do país;
- número da figurinha;
- status indicando se a figurinha já está na coleção.

## Dados enviados pelo usuário

O usuário pode enviar textos ou imagens de figurinhas para análise dentro do ChatGPT.

Essas informações são usadas apenas para:

- identificar códigos de figurinhas;
- comparar com a base oficial publicada no GitHub;
- indicar figurinhas novas, repetidas, duvidosas ou não encontradas;
- gerar sugestões de atualização da coleção.

## Armazenamento

Este GPT não armazena dados pessoais do usuário em sistemas próprios.

A base oficial da coleção fica armazenada no GitHub, dentro deste repositório.

## Compartilhamento de dados

Este GPT não vende, compartilha ou transfere dados pessoais para terceiros.

As Actions configuradas neste GPT consultam apenas arquivos públicos deste repositório no GitHub.

## Atualização da coleção

Este GPT não altera automaticamente o repositório GitHub.

Quando o usuário deseja adicionar figurinhas, o GPT pode gerar uma sugestão de atualização, mas a alteração final no arquivo do GitHub deve ser feita pelo próprio usuário.

## Dados pessoais

A coleção de figurinhas não deve conter dados pessoais sensíveis.

Recomenda-se não incluir nomes, documentos, endereços, informações financeiras, dados de saúde ou qualquer outro dado pessoal na base da coleção.

## Contato

Em caso de dúvidas sobre esta política, utilize os canais de contato disponíveis no perfil do proprietário deste repositório no GitHub.

## Alterações nesta política

Esta política pode ser atualizada conforme a evolução do GPT ou da estrutura da coleção.
